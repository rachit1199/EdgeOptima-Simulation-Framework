📚 21CSP302L – Minor Project: Energy-Efficient Task Offloading

Students: Rachit Yadav (RA2311003011471) & Ashray Ahuja (RA2311003011447)

Guide: Mrs. V. Sujatha, Assistant Professor (Jr.G), Department of Computing Technologies

This simulation demonstrates intelligent task offloading strategies in hybrid edge-cloud environments. The system optimally decides task placement across local devices, edge servers, and cloud datacenters to minimize energy consumption while maintaining Quality of Service (QoS) requirements.
